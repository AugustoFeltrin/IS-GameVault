# 🎮 GAMEVAULT

GameVault é uma plataforma social desenvolvida para jogadores registrarem seus jogos finalizados, avaliarem experiências, criarem listas temáticas e acompanharem estatísticas pessoais de gameplay.

Com uma interface moderna e intuitiva, o sistema busca transformar o acompanhamento da vida gamer em uma experiência social e organizada.

## 🕹️ Funcionalidades Principais

- **Registro de Jogos**: Marque jogos como finalizados e mantenha um histórico pessoal de gameplay.
- **Sistema de Avaliações**: Avalie jogos utilizando notas e contribua para a média geral da plataforma.
- **Wishlist**: Salve jogos que deseja jogar futuramente em uma lista personalizada.
- **Listas Temáticas**: Crie listas públicas como “Melhores RPGs” ou “Favoritos da Nintendo”.
- **Estatísticas de Perfil**: Visualize horas jogadas, média de notas e quantidade de títulos zerados.
- **Busca e Exploração**: Encontre jogos por nome, gênero, plataforma e popularidade via API.
